## Команды

#### Развертывание
```shell
make install
```
#### Сборка пакета
```shell
make build
```
#### Установка пакета
```shell
make package-install
```
#### Проверка линтером
```shell
make lint
```
#### Автоисправление линтером
```shell
make lint-fix
```
#### Форматирование кода
```shell
make format
```
